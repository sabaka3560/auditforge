🧠 Amazon Bedrock — Audit Narration
Calls Claude via Bedrock after each report generation. Translates raw audit rows into plain-English summaries: “3 BUs missing Locator Alias — likely caused by Oracle Fusion patch 24B changing the export column name.” Results cached in ElastiCache per report hash. Data never sent to Anthropic training. Tenant opt-in toggle in admin console.
📈 SageMaker Endpoint — Anomaly Detection
Isolation Forest trained on rolling 90-day history per tenant. Flags statistically anomalous audit results: “EMEA_MFG has 40% more Control Gaps than the same quarter last year.” Endpoint auto-scales to zero when idle. Retrained weekly via SageMaker Pipeline. Feature store backed by S3 + Athena.
